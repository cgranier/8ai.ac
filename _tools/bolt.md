Bolt.new is StackBlitz's AI-powered app builder that generates, runs, and deploys full-stack web applications entirely in the browser using WebContainers. It can scaffold projects across popular frameworks, install dependencies, and let you iterate via chat -- all without a local development environment. It quickly became one of the most popular AI coding tools for rapid prototyping.
